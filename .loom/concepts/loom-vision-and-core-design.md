# Loom Vision and Core Design

## 愿景

让系统文档不再是死板的归档，而是随开发进程同步进化的「活的大脑」。

## 使命

通过 MCP 协议将 AI 对话转化为结构化的 Markdown 知识库，利用 Git 实现多端同步，构建免维护的团队集体共识。

## Slogan

> Loom: Weaving your ephemeral AI chats into a persistent system mind.
> 将转瞬即逝的 AI 对话，编织成可持续演进的系统记忆。

## 核心价值

- **伴随式理解**：无需专门写文档，AI 在对话中自动沉淀知识
- **零成本运行**：利用 VS Code / Cursor 自带 AI 能力，无需额外 API Key
- **Git 原生**：所有记忆均为 MD 文件，支持版本回溯与 GitHub 协同
- **透明可控**：知识存储在本地，开发者随时可查可改

## 核心隐喻

「Loom」= 织布机。将零散的对话（线/Threads）编织（Weave）成结构化的知识绸缎。

## 核心循环

对话 → 提取知识 → 结构化更新 → 检索增强（RAG）→ 下一轮对话更精准
