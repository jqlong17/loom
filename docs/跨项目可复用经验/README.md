## 与本仓库的关系

上述模式在 Loom 中的落地同时依赖：

- **OpenCode** 作为可插桩宿主（源码版合并上下文日志后还可落盘 `requests.jsonl`）；
- **Loom** 作为 stdio MCP 服务端。

换宿主时只需替换「如何非交互发起一轮对话」的命令与配置合并规则，**沙箱与 MCP 侧思路不变**。
